连通性测试

1. 验证接口与路由状态

比如display vlan 2

能看到UT:GE0/0/1就没问题

2. 分层连通性测试

  1. 测试交换机互联(SWA -> SWB):ping ipv6 2001:db8:12::2
  2. 测试本地PC可达性:
    • SWA -> PC1:ping ipv6 2001:db8:101::50
    • SWA -> PC2:ping ipv6 2001:db8:102::50
    • SWB -> PC3:ping ipv6 2001:db8:103::50
    • SWB -> PC4:ping ipv6 2001:db8:104::50
  3. 测试跨交换机路由:
    • SWA -> PC3:ping ipv6 2001:db8:103::50
    • SWA -> PC4:ping ipv6 2001:db8:104::50
    • SWB -> PC1:ping ipv6 2001:db8:101::50
    • SWB -> PC2:ping ipv6 2001:db8:102::50
  4. 测试全网互通:
    • PC1 -> PC2:ping 2001:db8:102::50
    • PC1 -> PC3:ping 2001:db8:103::50
    • PC1 -> PC4:ping 2001:db8:104::50

注意：ENsp的模拟PC的ping命令无需加上ipv6字段可以直接ping ipv6的地址
